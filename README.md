## 📋 About rl-explainer

rl-explainer is an easy-to-use Windows app designed to help users learn the basics of reinforcement learning. It breaks down complex ideas into simple steps and shows examples you can follow. You do not need any programming skills or prior knowledge about machine learning to use this software.

## ⚙️ System Requirements

- Windows 10 or later  
- 4 GB RAM minimum (8 GB recommended)  
- 500 MB free disk space  
- An internet connection is helpful but not required after downloading  
- Screen resolution of 1366x768 or higher  

## 🛠️ Main Features

- Clear explanations of core reinforcement learning concepts  
- Interactive examples to guide your learning  
- Visual aids like charts and step flows  
- Simple user interface for easy navigation  
- Offline use after initial download

## 🎯 How to Use rl-explainer

This app organizes reinforcement learning topics in simple sections:

- **Basics**: Learn what reinforcement learning is, how it differs from other types of learning, and its real-world uses.  
- **Actions and Rewards**: Understand how actions affect results and how rewards guide learning decisions.  
- **Policies and Strategies**: Explore how rl-explainer shows decision-making rules in easy terms.  
- **Examples**: Step through visual examples that show how agents learn by trying different actions.  
- **Glossary**: A helpful list of terms with plain-English definitions.

Navigate by clicking on each section from the main menu or sidebar. Each page provides clear explanations and examples with helpful images.
